**Status:** ✅ **Shipped (v1.4)** — Feature-complete. Stretch goals pending.

Android gamified productivity app. Track habits, complete daily quests, and level up your character.

## Features

- **Daily Quests** — Random exercises with XP rewards
- **Habit Tracking** — Add, reorder, and track custom habits
- **RPG Progression** — XP, levels, ranks (F to SSS), character classes
- **Prayer Times** — Location-based calculation with multiple Islamic methods
- **Customizable Themes** — 10 color schemes with instant apply
- **Streak System** — Build momentum with daily consistency rewards
- **Passcards** — Skip days without breaking your streak

## Tech

- Minimum SDK: 24 (Android 7.0)
- Target SDK: 36
- Kotlin 1.9+
- Jetpack Compose
- Material 3 Design
- MVVM architecture with StateFlow
